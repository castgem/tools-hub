# Browser Tools Hub

Build one Astro site for a single production domain that combines the strongest grouped tool families already prototyped in this workspace.

Current launch scope:

- `dev-tools/`
  - `json-convertor`
  - `text-formatter`
  - `modern-devbox`
- `media-tools/`
  - `image-optimizer`
  - `webp-to-png`
  - `video-to-gif`
  - `logo-generator`
- `creator-tools/`
  - `social-profiler`
  - `xhunter`
  - `writetune`
  - `prompt-refiner`
- `document-tools/`
  - `pdf-easier`

Core product rule:

- One shared domain
- One shared trust layer
- One shared deploy target
- Clear category directories
- One narrow search intent per child page
