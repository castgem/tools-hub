## Tools Hub

This directory is the single deploy root for the shared-domain tools site.

- Repo root: this folder
- Build command: `npm run build`
- Publish directory: `dist`
- Main routes:
  - `/`
  - `/dev-tools/`
  - `/media-tools/`
  - `/about/`
  - `/contact/`
  - `/privacy/`

The older grouped projects in `projects/dev-tools-hub` and `projects/media-tools-hub` are now source references, not deploy targets.
