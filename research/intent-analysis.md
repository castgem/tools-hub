# Intent Analysis

## Root

- Dominant intent: commercial + navigational
- Goal: help users choose the right tool family quickly

## Dev Tools

- Dominant intent: transactional / BOFU
- Users want to solve JSON, text, regex, Base64, cron, or SQL tasks immediately

## Media Tools

- Dominant intent: transactional / BOFU
- Users want to compress, convert, or create assets immediately

## Creator Tools

- Dominant intent: transactional + commercial / BOFU
- Users want to preview, draft, humanize, or refine publishing-related output immediately

## Document Tools

- Dominant intent: transactional / BOFU
- Users want to merge, compress, split, rotate, or convert files immediately

## Structural Implication

- Do not flatten all tasks into one mega-tool homepage.
- Use the homepage to route traffic.
- Use category pages to preserve family-level topical relevance.
- Use child pages to convert direct intent.
